# SpellPod Brand Direction

## Theme
- Direction: `Arcane Tactical`
- Product personality: strategic, clean, focused, trustworthy
- UX principle: clarity over ornament

## Design Intent
SpellPod should feel like a precision deckbuilding workstation with subtle magical identity.  
The interface should never feel cluttered, noisy, or game-like at the cost of usability.

## Core Principles
1. Clarity First
- Prioritize readable hierarchy and obvious actions.
- Reduce visual noise in data-heavy screens.

2. Tactical Structure
- Use consistent layout grids and spacing.
- Keep interaction patterns predictable.

3. Restrained Arcane Flavor
- Express identity through palette and typography.
- Avoid excessive effects, glows, animations, or decorative clutter.

4. Accessibility by Default
- Maintain contrast and focus visibility in both themes.
- Ensure keyboard-friendly interaction flow.

## Theme Modes
- Light mode is default.
- Dark mode is fully supported.
- User preference is persisted locally.

## Typography Direction
- Headings: `Cinzel` (display serif)
- UI/Body: `Inter` (clean sans-serif)

## Anti-Goals
- Do not over-style every component.
- Do not rely on color alone to indicate state.
- Do not mix unrelated visual styles per feature.
